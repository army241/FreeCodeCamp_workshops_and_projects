# Parent-Teacher Conference Form

A simple and elegant HTML/CSS form to help parents schedule conferences with teachers. This form collects student information, parent/guardian details, preferred contact method, and any additional notes.

## Features

- Collects **student name** and **grade**  
- Collects **parent/guardian name**  
- Choose **preferred contact method** (Email or Phone) with custom radio buttons  
- **Additional notes** section for concerns or topics to discuss  
- Responsive design with a modern, clean look  
- Styled entirely with **CSS** (no JavaScript required)
