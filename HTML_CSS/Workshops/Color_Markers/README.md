# CSS Color Markers

This project is a simple visual design created as part of a **freeCodeCamp workshop**.

It demonstrates:
- Basic HTML structure
- External CSS styling
- Use of RGB, HEX, HSL, and RGBA color formats
- Linear gradients
- Box shadows
- CSS class-based styling

## Technologies Used
- HTML5
- CSS3

## Purpose
Practice working with colors, gradients, and visual styling using pure CSS.
