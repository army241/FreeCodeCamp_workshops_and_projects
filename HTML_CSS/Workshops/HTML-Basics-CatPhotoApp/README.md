# Cat Photo App

This project is a basic HTML page created as part of a **freeCodeCamp workshop**.

It demonstrates:
- Basic HTML document structure
- Headings and paragraphs
- Links and images
- Lists (ordered and unordered)
- Semantic elements such as `main`, `section`, `figure`, and `footer`
- Image accessibility using `alt` text

## Technologies Used
- HTML5

## Purpose
Practice fundamental HTML concepts and semantic markup.
